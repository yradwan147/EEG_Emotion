\textbf{All 25+ V-axis-as-supervision interventions, ranked by $\Delta$.}
Each row is one variant tested in cycle 75; bar length is $\Delta$ FACED 9-class BACC vs the same-recipe vanilla baseline (paired-mean over $\geq 3$ seeds where available, single-seed for catastrophic failures).
Significance markers: $***$ $p{<}0.001$, $**$ $p{<}0.01$, $*$ $p{<}0.05$, ns otherwise (paired $t$-test).
Color tier: dark red = catastrophic destruction ($\Delta{<}{-}0.1$); red = significant negative; orange = NS negative; light blue = NS positive.
